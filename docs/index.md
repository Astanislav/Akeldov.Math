# Akeldov.Math

Akeldov.Math is a repository for .NET math libraries.

The solution keeps related libraries together so they can share conventions, tests, documentation, and release workflow.

## Libraries

| Library | Version | Description |
| --- | --- | --- |
| [Akeldov.Math.Spatial2D](Spatial2D/) | `0.1.0` | Two-dimensional geometry, vector math, spatial sampling, partitioning, and influence field utilities. |
| [Akeldov.Math.Intervals](Intervals/) | Not packaged | Interval primitives for numeric ranges. |
