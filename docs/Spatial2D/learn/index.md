# Learn

This section covers the main building blocks in Akeldov.Math.Spatial2D.

Start with vectors and curves for core geometry operations, then move to sampling, partitioning, and influence fields for higher-level spatial workflows.

## Topics

- [Vectors](vectors.md) for 2D coordinates and vector math.
- [Curves](curves.md) for lines, rays, segments, circles, arcs, projections, and intersections.
- [Poisson Disk Sampling](poisson-disk-sampling.md) for generating well-spaced point sets.
- [Voronoi Partitioning](voronoi-partitioning.md) for assigning positioned items to weighted sites.
- [Influence Fields](influence-fields/) for sampling values from point and curve sources.
